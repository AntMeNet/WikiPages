Creating an Ant Colony

In this lesson we'll discuss how to get started with the game itself by creating an ant colony and placing them in their virtual world.

The following instructions and screenshots are all based on the Visual Studio 2013 development environment as well as the .NET Framework 4 and .XNA frameworks. If you're using a different development environment, things may look a bit different.

Quick reminder: .NET Framework 4 makes it possible to write and run programs that have been developed with the C# or VB.NET programming languages. The .XNA framework is what makes the game's 3D view possible.

Our goal here is to get you into the game as quickly and easily as possible. For that reason we're not going to provide a full explanation of all the tabs and menus on the start screen (don't worry, though, we'll cover them in depth later). You don't need all that info to start playing the game. You can come back to the lessons later if you want to read up on the advanced features.

OK, so what happens now?

In the following steps we're going to create our first ant colony together. The new ant colony with its tiny citizens is your very own programming project. The behavior of the ant colony depends on the commands you give it by writing and editing the program.

On the start screen you'll find three tabs above the greeting: "Welcome", "Statistics" and "3D View". Above the tabs you'll find the menu bar with the "Mode" and "Speed" menus. You'll also see the "Play", "Stop" and "Pause" symbols up there on the left. They'll all be gray because you can't click them yet.

Click the "Mode" button and select "Free Game" by right-clicking. A new tab labeled "Free Game" will appear in the tab bar.

Click the new "Free Game" tab and then click one of the empty slots in the menu that appears.

The "Select Player" window will appear. You'll see icons for various templates for different kinds of ant behavior. They'll be important later, but for now we just want to create our own ant colony.

To do that, click on the “+” symbol in the menu bar and the “Create Ant Colony” window will open.

Give your new colony project a name. Note: the name can only contain letters and numbers, it can't begin with a number, it has to be at least two characters long and it can't be more than twenty characters long. We named our colony “Alpha”. When you're done, click “OK” and the “Edit Ant” window will open.

This window is confirmation that you've created a new ant template. Click “Yes” to open your development environment (in this case, Microsoft Visual Studio). You'll need it to be able to configure and control your colony by writing and editing source code for the program that “runs” the colony.

Wow, that's a lot of windows and menus and buttons! That's because Microsoft Visual Studio is a highly developed tool for professional programmers. With the help of development environments like this one, programmers have made every kind of software you can imagine, including the Windows operating system that you're using right now. But don't panic! We only need to work with a couple of these elements: the big window in the middle (the Code Editor) and the little window on the right (the Solution Explorer).

Now we're ready. Click the start button (it's the green triangle) to load your newly created colony.

The AntMe! start screen that you know and love will appear again. You'll notice that the “Play” and “Pause” symbols above the greeting are now clickable.

Click “Play” to start the simulation and bring your ants to life in the 3D view.

Now two AntMe! windows are open: the 3D window and the start screen. Check out the “3D View” tab on the start screen to see how you can control the 3D view and move the camera around the sandbox.

Move the camera with the mouse in order to explore the inner life of the sandbox. Take your time and have a look around in order to get a feel for the game world and its inhabitants.

If you move the camera close to the anthill, you'll be able to see your ants' feelers poking out of the anthill.

Why aren't the ants moving? Well, we created the Alpha colony and the anthill has been populated with ants. But we still haven't told the ants what they should do or how they should behave. We'll cover that in the next lesson. You'll find out how you can get your ants to move around and how to get them to gather food.
